# Web3 NFT Airdrop Engine

A professional Web3 reference project that demonstrates how NFT airdrop systems can be modeled using clean and modular Web3 architecture.

This repository focuses on simulating eligibility rules, distributing NFTs, tracking claims, and generating airdrop analytics in a beginner-friendly and production-style format.

---

## Supported Networks

- Ethereum  
- Base  
- Polygon  
- Arbitrum  

---

## Project Goals

- Simulate NFT airdrop eligibility  
- Track NFT distribution flow  
- Monitor claim status  
- Generate airdrop analytics  
- Teach NFT distribution architecture  

---

## Architecture Overview

This project uses a root-only structure where each file handles a single responsibility such as network access, user registry, eligibility engine, NFT distributor, and analytics reporting.

---

## Features

- Multi-chain RPC support  
- NFT eligibility simulation engine  
- NFT distribution logic  
- Claim tracking system  
- Clean console-based reporting
